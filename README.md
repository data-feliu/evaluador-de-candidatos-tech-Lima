# Evaluador de Candidatos Tech — Lima

Sistema en Python que evalúa el perfil de un candidato
para roles en Data Analytics, Data Science y MLOps.

## ¿Qué hace?
- Recibe datos del candidato por consola
- Calcula un puntaje sobre 100 puntos
- Determina roles para los que aplica hoy
- Da recomendaciones priorizadas
- Proyecta tiempo hasta cada objetivo

## Tecnologías
- Python 3.14
- Condicionales, operadores lógicos, f-strings
